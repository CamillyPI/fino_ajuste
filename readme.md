## 4 - Como rodar:
#### 1 - Download:
~ Baixe o zip, descompacte e abra na sua IDE.

#### 2 - Crie o ambiente virtual:
~ python -m venv venv

#### 3 - Inicialize o ambiente:
~ source venv/Scripts/activate

#### 4 - Instale as dependências:
~ pip install -r requirements.txt

#### 5 - Inicialize o backend
~ uvicorn backend.main:app --reload

#### 6 - Abra o arquivo index.html no navegador

#### 7 - Abra o Swagger UI: http://localhost:8000/docs

#### 8 - Teste os Endpoints :)
